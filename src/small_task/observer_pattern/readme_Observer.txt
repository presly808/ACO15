Дано:
----------------------

    Пакет weather - содержит реализацию Метое Станции с использованеем своей реалзиции паттерна Наблюдатель.
        -   Subject - интерфейс субъекта
                -   WeatherData - реализует интерфейса субъекта, Метео Станция
                    -   humidity    - влажность
                    -   temperature -   температура
                    -   pressure    -   давление
        -   Observer - интерфейс наблюдателя
        -   DisplayElement  -   интерфейс для реализация отображения объектов

    Визуальные элементы: (Они же наблюдателя. Все наблюдатели реализует интерфейс Observer и регистрируются у субъекта)
        -   HeatIndexDisplay - отображает тепловой индекс
        -   StatisticsDisplay - вычисляет и выводит минимально, текущее и среднее значение
        -   ForecastDisplay - выводит прогноз погоды по показаниям барометра
        -   CurrentConditionsDisplay - выводит текущие значения

    WeatherStation  -   Тестовый класс
    WeatherStationHeatIndex -   Тестовый класс кооторый задействует визуальный элемет HeatIndexDisplay

    Пакет weatherobservable - содержит реализацию Метео Станции но с использованием встроенного в Java API паттерна
наблюдатель.
    Пакет swing -   содержит пример использования паттерна Наблюдатель с визуальными компонентами Swing.
